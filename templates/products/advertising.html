Smart Advertising

The application of this project can be in retail places such as malls, shopping complexes and airports (only for retail purposes). The process starts with the user scanning the QR code in the entry of the mall/ any other retail place. The QR code leads the user to a Web Application where all the details of the mall are displayed. 1. Contact information such as helpdesk number, general mall inquiry, mall security, mall parking etc. 2. An attached map for the user to know about the directions 3. Different categories such as food, clothing, footwear, entertainment, electronics, groceries etc. 4. Each category has the information such as the list of the shops present along with the level numbers. 5. The virtual experience page helps the user to explore the mall virtually with going. This is done using Google Street View API.
